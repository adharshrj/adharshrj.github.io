Material Classifying Hexa-tron Robot

Project information

  • Category: Robotics/Machine Learning
  • Client: SRM Institute of Scinece and Technology
  • Project date: January 2020 â€“ April 2020
  • Award: Graduation Project

ROLE:-
Acting as the leader, using python coding in the raspbian environment with a tensorflow model, a robot that classifies materials as metals, plastics, rubber and so on based on a dataset of 2000 images for each material was designed. The different parts of the robot were designed in CAD and then a CNC router was used to cut out the parts from acrylic sheets.

LANGUAGE USED:-
Python

PACKAGES USED:-
OS, Agparse, Time, PIL image,(OPENCV)CV2, Numpy, Tensorflow
