Catagory: Application

Client: NASA

Date: October 2020

Award: Global Nominee

Link: Click Here

ROLE:

The goal was to develop an application that will assist astronauts in space to help them maitain a proper dietary intake and perform daily exercise to maintain peak performance in zero gravity situations. I performed a complete analysis on the dataset provided by NASA to derive the essential parameters. The essential parameters were Basal Metabolic Rate or BMR of the individual in zero gravity situations and nutrient partioning advised by NASA.

Based on my research we designed a preliminary web application, the application was deployed via IBM Watson Cloud. Then I ported the web app into an android app using Android SDK.

We cross checked the parameters and submitted after finalization. I wanted the application to be as light as possible so that we can port them to smart watches in the future.

LANGUAGES USED:
Python, Java, Kotlin, Javascript, HTML, CSS
