SMART ASSISTANT

Catagory: Application

Client: NSD

Date: Febraury 2019

Award: Best Project On NSD

ABOUT:

ABSTRACT: A personal assistant system to serve the differently abled people of the society.

ROLE: Using TTS engine and AWS api, I created a virtual assistant for the differently abled people of the society.

The smart assistant system successfully communicated with the with the user and performed the tasks requested. The project was sent to the National Science Day project competition organised by the Indian Science Congress Association and was awarded the best project award.

LANGUAGE USED: Python


API USED: AWS Api
